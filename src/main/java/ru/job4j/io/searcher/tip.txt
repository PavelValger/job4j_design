Ошибка ввода! Ниже приведены правила работы программы!

Пример запуска программы: "java -jar find.jar -d=c:/ -n=*.?xt -t=mask -o=log.txt".
Ключи:
-d - директория, в которой начинать поиск.
-n - имя файла, маска, либо регулярное выражение.
-t - тип поиска: mask искать по маске, name по полному совпадение имени, regex по регулярному выражению.
-o - результат записать в файл.

Имя файла должно включать расширение файла.

Маска и регулярное выражение должны соответствовать правилам написания маски и шаблону регулярного выражения.
Для задания масок кроме обычных символов, допустимых в именах файлов, используют специальные символы:
"*" - заменяет любое количество (в том числе и нулевое) любых символов;
"?" - заменяет один и только один обязательно стоящий в указанном месте символ.
"." - после символа "дот" следует расширение искомого файла.
Документация по регулярным выражениям - https://www.oracle.com//RegularExpression.html

Тип поиска должен быть строго mask, name или regex.

Результат записывается строго в файл с расширением ".txt".
